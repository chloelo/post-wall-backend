# Post Wall

此為會員制的貼文牆後端 API 設計，僅供練習，無任何商業用途。



## 使用技術 🔧
- 語言：Node.js
- 框架：Express
- 資料庫：MongoDB
- 雲平台：Vercel

### Installation 📦
```sh
cp .config.env.example .config.env
npm install
```

### Usage 🚀
```sh
npm run start:dev
```

### API 測試 💻 
請開啟 postman 後載入以下檔案即可
- post-post-wall.postman_collection.json
- post-wall.postman_environment.json
